### 브랜치 전략
- 기능별 브랜치 생성(ex. feat-FE-oauth)
- 기능 개발 완료에 따라 분야별 브랜치에 병합(ex. feat-FE-oauth -> FE-dev)
- 코드리뷰 완료에 따라 대표 브랜치에 병합(ex. FE-dev -> team-5)
